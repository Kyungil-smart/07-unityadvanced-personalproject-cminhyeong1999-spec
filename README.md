# Run away from Isekai Truck!!

0. 소개
   분명 길을 걸어가고 있었는데 잠깐 눈을 감았다 뜨니 주위에 이상한 트럭들 밖에 없다... 설마 이거에 치이면...

   무기를 고르고, 다방면에서 쏟아져 나오는 적들을 피해 경험치를 먹고, 일정 경험치를 먹어서 레벨업을 하여 버티자

   - 프로젝트 이름 : Run away from Isekai Truck
   - 장르 : 2d 슈팅 게임
   - 플레이타임 : 20분 예정
   - 개발 환경 : Unity 6000.3.9f1
   - 플랫폼 : Windows
   - 프로젝트 기간 : 8일
   - 제작자 : 조민형

1. 플레이 방법

   [조작법]
   - 상하좌우 방향키로 이동 가능 + 대각선 이동 가능
   - 자동 공격

   [스테이지]
    - 끝 없는 스테이지 에서 20분 동안 생존
    - 일정 시간이 지나면 스테이지에 나타는 몬스터 변화

  [무기]
   - 플레이어를 중심으로 도는 도끼 / 근접 1종류
   - 전방을 향해 총알을 발사하는 총 , 일정 범위를 탐지하여 적을 쫒는 미사일 / 원거리 2종류
